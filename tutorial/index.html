    Introduction

    In this tutorial, we’ll cover the absolute basics of Python:

    • Variables: how to store values.
    • Functions: how to group reusable logic.
    • Loops: how to repeat actions.
    • Conditionals: how to make decisions.
    • Lists: how to work with ordered collections.

    Every code snippet below is fully editable—just click Run to see the result.

    Functions
    

    Functions let you package a set of statements and reuse them by name. They can accept parameters and return values.
    



    

    

    




    Explanation
    

      

    1. def greet(name): starts a function definition called greet, with one parameter (name).  
      2. 

    2. return f"Hello, {name}!" uses an f-string to build a greeting.  
      3. 

    3. We call greet("Alice"), store it in message, and then print it.

    Loops
    

    Loops let you repeat a block of code multiple times. The most common are for-loops and while-loops.
    



    

    

    




    Explanation
    

      

    • for i in range(3): repeats the indented block with i = 0, 1, 2.  
      • 

    • while count > 0: repeats as long as count is greater than zero. Inside, we decrement count.
      • 

    

    

    Conditionals
    

    Conditionals let you execute code based on boolean expressions (if, elif, else).
    



    

    

    




    Explanation
    

      

    1. if x < 5: checks if x is less than 5.  
      2. 

    2. elif x == 5: checks if x is exactly 5.  
      3. 

    3. else: handles all other cases.  
      4. 

    4. Change x to other values to see different branches run.
      5. 

    

    

    Lists
    

    Lists are ordered collections that can hold items of any type. You can index them, append items, and iterate through them.
    



    

    

    




    Explanation
    

      

    • fruits = ["apple", "banana", "cherry"] creates a list of strings.  
      • 

    • fruits[0] accesses the first element.  
      • 

    • fruits.append("date") adds a new element at the end.

    • The for fruit in fruits: loop iterates over each element.
